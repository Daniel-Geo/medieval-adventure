# medieval-adventure
​You were captured in a medieval prison and you need to escape

### Objective:

​Run as far as you can until you escape from the prison or fall in a trap :)

### Controls:
​Move: WASD keys.  
Jump: Space key.  
Look: Mouse movement.  
Free-look: Press Alt key while moving the mouse to look in one direction while walking in another.  
Sprint/Run: Press Shift key while walking.  
Crouch: Q key.  
Slide: Press Shift + Q while running.  

## Built with
[Godot Game Engine v4.6.stable](https://godotengine.org)

## Why I built this project
I have been learning game development using Godot for about a year now, but I have never built a 3D game using it before, so I wanted to to learn about 3D game development concepts in Godot and develop a game using it.  
I liked the experience of developing a 3D game.
